# 项目定位与核心研究问题

## 项目定位

本项目不叫「Prompt 打分器」，而叫：**AI 交互素养评估原型**。

核心目标：在自然、轻松的任务对话中，评估用户与 Agent 协作解决问题的能力（有效、批判、负责地参与 AI 系统）。

## 研究问题

1. AI 素养能否被操作化为「自然对话中的 Agent 协作能力」？
2. 能否通过「按画像分场景」提升评估的真实感，同时保持统一的核心 rubric？

## 两天范围（MVP 边界）

- 基础版做「按用户分场景」，不做「按用户改核心评分定义」。
- 核心 rubric 固定，场景按用户画像变化，反馈措辞可变化。
- 交付：可运行的 4 场景 + 事件记录 + 单 judge 评分 + 规则校正 + 基础反馈展示。

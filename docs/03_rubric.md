# 最终评分维度 (Rubric)：5 维定义与 0–5 Anchor

评估采用统一的核心 Rubric：5 个维度，每维 0–5 级，内部按权重换算为**百分制总分（0–100）**展示；各维度在结果页也以 0–100 显示（level × 20）。

---

## 评分等级 0–5（通用 Anchor）

| Level | 含义     | 行为锚定 |
|-------|----------|----------|
| 0     | 完全缺失 | 该维度无任何可观察行为或明确意图。 |
| 1     | 很弱     | 仅有极少量、被动或模糊的体现，难以支撑任务。 |
| 2     | 较弱     | 有体现但不完整、不连贯或经常依赖提示。 |
| 3     | 基本可用 | 能覆盖主要要求，偶有遗漏或不够主动。 |
| 4     | 较强     | 主动、连贯、多数情境下表现稳定。 |
| 5     | 非常成熟 | 系统化、有边界意识、可复现的高质量协作。 |

以上 anchor 适用于下面每个维度，具体打分时结合维度含义与对话证据。

---

## 五维度与权重（总和 100）

1. **Clarity (说清任务)** — 40%  
   是否把目标、约束、受众/对象说清楚；有无模糊或遗漏导致 Agent 误解。

2. **Context (补足上下文)** — 15%  
   是否在需要时补充背景、前提、例子或限制，而不是默认 Agent 全知。

3. **Steering (推进对话)** — 20%  
   是否在跑偏时拉回、在不足时追问、在有多解时引导方向。

4. **Judgment (判断结果)** — 15%  
   是否对 AI 输出做核实、比较、取舍，而不是照单全收（用于区分「被 AI 带着走」）。

5. **守住边界** — 10%  
   是否注意敏感信息、权责边界。

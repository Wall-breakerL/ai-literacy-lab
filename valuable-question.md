# Valuable Questions

这个文件记录项目推进中值得保留的高价值讨论、设计判断和待办问题。后续所有有深度的问题都可以继续追加到这里，避免散落在聊天记录里。

## 2026-04-28

### 1. AI-MBTI 应该是“访谈定制问卷”，不是纯聊天判断

结论：当前产品主线定为“轻量访谈收集背景 + Agent B 生成定制问卷 + 服务端确定性计分 + LLM 解释报告”。

原因：
- 纯聊天判断更自然，但稳定性和可解释性弱。
- 问卷能提供更可控的评分依据。
- 访谈负责把用户职业、近期场景、目标带入题目，避免问卷泛化。

### 2. Agent A 收集目标，但不能让目标成为流程门槛

结论：Agent A 开场同时问身份、AI 使用经历、当前目标；第二轮只补问一次目标。到轮次后无论目标是否完美，都进入问卷。

原因：
- 如果要求 A 必须问清楚目标，模型能力、用户短回答、网络重试都会让流程卡住。
- 目标缺失时可以降级绑定近期 AI 使用场景。

### 3. 问卷需要“不了解 / 没想好”

结论：这是独立选项，不作为第 7 档分数。选择后该题跳过，不进入平均分。

原因：
- “中间分”代表中性或摇摆。
- “不了解 / 没想好”代表用户缺少经验或暂时无法判断。
- 两者不能混在一起计分。

### 4. 16 题先保持，32 题放到下一阶段

结论：MVP 保持 16 题，每维 4 题；先把跳过题、正反向校验、服务端计分做好。32 题放到正式量表阶段再做。

下一阶段再讨论：
- 是否升级为 24 或 32 题。
- 是否用自适应补题：某维跳过多或分数接近 50 时追加题目。
- 是否在 32 题阶段引入更严格的低信心度判定。

### 5. 低信心度暂时只记录，不影响人格画像

结论：目前先记录 `answeredCount / skippedCount / confidence`，但暂不让低信心度改变 16 型画像判定。低信心度参与“暂定画像”的设计，放到 32 题阶段处理。

原因：
- 当前 16 题样本较短，过早引入复杂判定会增加解释负担。
- 等题量扩充后，再把低信心度和人格 code 的不确定性绑定更合理。

### 6. 报告结构：先解析，再建议

结论：报告顺序为“画像概览 -> 四维解析 -> 整体建议 / prompt 模板”。

原因：
- 用户应该先理解为什么得出这个结论，再看如何改进。
- 每个维度只解释依据，不在每张维度卡里塞建议，避免空洞和重复。

### 7. 头像要强参考 MBTI 官方图片的识别语言，但不能直接复刻

结论：头像生成 prompt 强调“official MBTI / 16Personalities result-avatar aesthetics and product language”，但要求原创人物、不同服装/姿势/脸/道具组合。

原因：
- 高度一致的人格测试头像能降低用户理解成本。
- 直接复刻官方图会有版权和品牌风险。
- 生成后需要人工审核，避免过像官方角色。

### 9. AI-HQ 下一版：标准化访谈 + 证据化评分 + 行动型报告

结论：AI-HQ 保持 5 段固定访谈，但服务端根据用户回答数确定当前阶段，不再依赖 Agent A 自己记顺序或输出结束标记。Agent B 只在报告阶段通读 transcript，判断 probes 与生成解释，分数和等级由服务端确定性计算。

原因：
- 固定 5 段访谈更适合能力测评的可比性。
- 让模型同时判断证据、计算分数和写报告会降低稳定性。
- 报告需要给出行动建议和可复制 prompt 模板，才有用户可感知价值。

本轮取舍：
- 题目改成跨领域通用，不再默认 coding 场景。
- 先做确定性计分，不完整引入 N/A / insufficient evidence。
- 保持 L1-L3，不扩展 L4。

### 10. Phase 3 取舍更新：AI-HQ 暂存，产品主线聚焦 AI-MBTI

结论：AI-HQ v0.1 暂时从首页主入口和主测试链路中下线，路由和代码保留为 archived。下一阶段优先把 AI-MBTI 的单 Agent、SessionState memory、反馈闭环做好。

原因：
- 当前用户体验和产品叙事应集中在 AI-MBTI 主线。
- AI-HQ 独立五段访谈会分散入口注意力，也和未来的 memory/evidence 机制重复。
- 更合理的方向是把 AI-HQ 能力成熟度改成 AI-MBTI 报告里的补充模块，使用同一段访谈和同一套 evidence。

后续回归条件：
- AI-MBTI 已有稳定 SessionState。
- 报告能引用访谈 evidence。
- 能力成熟度不再以独立考试感呈现，而是作为行动建议的一部分。
